문제2 통신장비에 채팅 기능 추가
---

## 주요 기능

* **멀티스레드 서버**: 여러 클라이언트 동시 접속 처리
* **입/퇴장 브로드캐스트**: `닉네임님이 입장하셨습니다.` / `퇴장하셨습니다.`
* **전체 채팅**: 모든 접속자에게 전송 (`닉네임> 메시지`)
* **귓속말**: `@닉네임 메시지` 또는 `/w 닉네임 메시지`
* **종료**: 클라이언트에서 `/종료` 입력 시 연결 종료
* **닉네임 표기 규칙 (클라이언트 화면)**

  * **내 메시지**: 내용만 표시(닉네임 생략, 오른쪽 정렬)
  * **남 메시지**: `닉네임> 메시지` (왼쪽 정렬)
  * **남의 귓속말**: `닉네임(귓)> 메시지` (왼쪽 정렬)
